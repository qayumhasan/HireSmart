# HireSmart

**HireSmart** is a Laravel-based job portal platform that connects employers with qualified candidates efficiently. Employers can post job openings and hire the best talent, while candidates can browse jobs and apply directly.

### Step 1: Clone the Repository

```bash
git clone git@github.com:qayumhasan/HireSmart.git
cd hiresmart
git checkout master
```

---

### Step 2: Laravel Setup

1. Rename the environment configuration file:

   ```bash
   mv .env.example .env
   ```

2. Copy necessary environment variables from `docker-compose.yml` to your `.env` file:

   Example:

   ```
   APP_URL=http://127.0.0.1
   ```

3. Install backend dependencies:

   ```bash
   composer install
   ```

4. Install frontend dependencies:

   ```bash
   npm install
   ```

5. Build frontend assets:

   ```bash
   npm run dev
   ```

---

### Step 3: Docker Setup

Make sure [Docker](https://www.docker.com/) is installed and running on your system.

Then run:

```bash
docker-compose up -d --build
```

Check running containers:

```bash
docker ps
```
